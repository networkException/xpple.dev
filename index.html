Frequently Anticipated Questions

What am I looking at?

You're looking at a command-line interface (CLI), which is an interface in which you can write and execute... commands! But this one isn't like the others, this one is made by me! It uses fairly simplistic JavaScript code along with some php which can be found here. The purpose of this CLI isn't anything serious. The commands are mostly for entertainment.

Do you keep track of the commands I execute?

No, with a few exceptions. Since I mostly use Javascript, the commands you execute are executed on your end. However, some commands require server-sided code. This means the php requests will be visible to me. Keep in mind that the results of any command will never be visible to me, as they are presented to you with JavaScript. For transparency, you may click here to see the server-sided code.
